#### Coin app using React.js and Redux Toolkit.

</div>

### Tools

- React.js
- Redux Toolkit
- SASS / SCSS
- React Icons
- Chart.js for market price changes
- React Router V6 
- Axios for API calls
- Moment for convert date time
- React-Toastify for notification
- React-loader-spinner for loading
- Redux-logger for actions
